# pendeteksi-uang-rupiah-yolov8
Computer Vision Project using YOLOv8 to Detect Indonesian Rupiah Banknotes
# 💵 Pendeteksi Jenis Uang Rupiah Menggunakan YOLOv8

## Deskripsi

Proyek ini merupakan implementasi Computer Vision menggunakan algoritma YOLOv8 untuk mendeteksi nominal uang kertas Rupiah Indonesia.

Model dilatih menggunakan dataset dari Roboflow Universe dan dijalankan pada Google Colab.

---

## Dataset

Dataset berasal dari Roboflow Universe.

Jumlah gambar:
- ±1.100 gambar

Jumlah kelas:
- Rp1.000
- Rp2.000
- Rp5.000
- Rp10.000
- Rp20.000
- Rp50.000
- Rp100.000

## Instalasi

Install library

```bash
pip install ultralytics
pip install roboflow
pip install opencv-python
```

Clone repository

```bash
git clone https://github.com/bagosajalah/pendeteksi-uang-rupiah-yolov8.git
```

---

## Cara Menjalankan

1. Buka Google Colab
2. Upload notebook
3. Install library
4. Download dataset dari Roboflow
5. Training model
6. Upload gambar uang
7. Jalankan prediksi
